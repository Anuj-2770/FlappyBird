# Flappy Bird - Java Version

🎮 **Flappy Bird** is a classic 2D arcade game implemented in Java using **Swing** for graphics and **Timers** for game loops.

---

## Features

- Control the bird using the **SPACE** key.
- Avoid pipes and do not hit the top or bottom boundaries.
- Collect coins for **bonus points (+10)**.
- Dynamic pipe placement with random heights.
- Coins appear randomly on the screen (never on pipes).
- Background music and sound effects.
- High score tracking saved locally (`highscore.txt`).

---

## Controls

- **SPACE**: Make the bird jump.
- **SPACE** (after game over): Restart the game.
